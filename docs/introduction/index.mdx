# 📘 Introduction

## Overview
Oraxen allows you to customize your Minecraft server by allowing the creation of new blocks, items, and recipes. 
This can be accomplished entirely through a resource pack and server-side implementation, eliminating the requirement for any client-side mod installations.

## How It Works
Oraxen reads your configuration files and uses them to generate a minecraft resource pack that will then be uploaded to a resource pack server which then sends the pack to players joining your server.
Polymath is a free and opensource software written in Python by Oraxen to host Minecraft resourcepacks. Oraxen supplies you with a free Polymath instance, hosted in Switzerland on an Oracle Virtual Private Server.

## Version
Oraxen is developed and designed for the <Highlight color="#1877F2" to="https://papermc.io/downloads/paper">**Paper**</Highlight> Server Software. It is compatible with most common forks including <Highlight color="#1877F2" to="https://papermc.io/software/folia/">**Folia**</Highlight>. Oraxen is also compatible with the <Highlight color="#FFA500" to="https://mcserverjars.com/spigot">**Spigot**</Highlight> Server Software, however our changes are not actively tested against it and issues e.g. with MiniMessage may occur.

Oraxen includes support for versions from **1.20.1** up to **26.2**. Oraxen has ViaVersion support, for that please read the documentation <Highlight color="#2E8B57" to="./compatibility/viaversion">**here**</Highlight>.

## Installation
Drag and drop the .jar file into your server's /plugins folder and restart your server.

<Callout type="note">
  Oraxen requires PacketEvents (or Protocollib) to function. Since Version 1.203.1, Oraxen automatically downloads PacketEvents when not already installed.
</Callout>
