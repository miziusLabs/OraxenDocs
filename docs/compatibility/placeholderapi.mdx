# 🔤 PlaceholderAPI

## Overview
Oraxen registers several PlaceholderAPI placeholders. This allows you to reference glyphs, shifts or the pack hash dynamically via placeholders.

## Usage
`%oraxen_yourglyph%` - Basic usage.<br />
`%oraxen_shift_N%` - Shifts N pixels to the right (positive).<br />
`%oraxen_neg_shift_N%` - Shifts N pixels to the left (negative).<br />
